
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%FUNCTIONALITATE%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
createTree:
  Creaza un arbore si celula dummy iar apoi seteaza link-urile acestora.
createTreeNode:
  Creaza un nou nod si seteaza link-urile atribuite acestuia.
destroyTreeNode:
  Sterge/elibereaza memoria arborelui primit ca parametru.
isEmpty:
  Verifica daca arborele este gol.
search:
  Cauta un element in arbore, daca este gasit este returnat.
minimum:
  Returneaza minimul de la un anumuit nod, acesta este reprezentat de cel
  mai mic copil frunza din arborele stang.
maximum:
  Returneaza maximul de la un anumuit nod, acesta este reprezentat de cel
  mai mare copil frunza din arborele drept.
successor:
  Reprezinta primul parinte care nu are copilul din dreapta null.
predecessor:
  Reprezinta primul parinte care nu are copilul din stanga null.
avlGetBalance:
  Retruneaza balansarea pe nodul respectiv ( H-STANGA - H-DREAPTA ).
avlFixUp:
  Roteste arborele in functie de balansare pentru al aduce la un arbore normal.
insert:
  Insereaza un element in arbore daca elementul nu este dublicat iar daca este
  atunci se insereaza in lista avand ca prev acelas element.
delete:
  Sterge un element din arbore daca nu este dublicat iar daca este dublicat 
  sterge prima dublura a acestuia.
destroyTree:
  Elibereaza memoria fiecarui nod, element, info iar apoi a arborelui.
